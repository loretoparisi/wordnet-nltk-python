# wordnet-nltk

## Build [docker]
```
docker build -f Dockerfile -t wordnet .
```

## Run [docker]
We run the Docker image with a attached `/src` volume for editable sources.

```
docker run --rm -it -d -v `pwd`/src:/app --name wordnet-test wordnet
docker exec -it wordnet-test bash
```
